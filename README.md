## 분석 내용
# Bank Customer Churn Analysis

## 프로젝트 소개

본 프로젝트는 은행 고객 데이터를 활용하여 고객 이탈(Customer Churn)에 영향을 미치는 요인을 분석한 프로젝트입니다.

고객의 국가, 성별, 연령 등 다양한 특성을 비교하여 어떤 집단에서 이탈이 많이 발생하는지 확인하고, SQL을 활용한 데이터 탐색 및 기초 분석 과정을 수행했습니다.

본 프로젝트는 데이터 분석의 전 과정보다는 데이터 이해(Data Understanding)와 탐색적 데이터 분석(EDA)에 중점을 두고 진행했습니다.

---

## 프로젝트 목표

* 데이터의 전체 구조와 특성을 파악한다.
* 고객 이탈 현황을 확인한다.
* 국가별 고객 이탈률을 비교한다.
* 성별에 따른 고객 이탈 차이를 분석한다.
* 연령대별 고객 이탈률을 비교하여 이탈 가능성이 높은 고객군을 확인한다.

## 데이터 탐색 (Data Exploration)

분석에 앞서 데이터의 기본 구조와 품질을 확인했습니다.

### 1. 데이터 규모 확인

* 전체 고객 수 : **10,000명**
* Customer ID 중복 여부 확인
* 모든 고객이 고유한 ID를 가지고 있는 것을 확인

### 2. 주요 변수 확인

분석에 사용한 주요 변수는 다음과 같습니다.

* Customer ID
* Credit Score
* Country
* Gender
* Age
* Tenure
* Balance
* Products Number
* Credit Card
* Active Member
* Estimated Salary
* Churn

### 3. 데이터 기초 확인

각 변수의 분포를 확인하고 고객 수를 집계하여 데이터가 정상적으로 저장되어 있는지 확인했습니다.

또한 국가, 성별, 연령 등의 범주형 변수는 GROUP BY를 활용하여 고객 수를 집계하였으며, 이후 이탈률 분석의 기준 데이터로 활용했습니다.

---

## 고객 이탈률 분석

먼저 전체 고객 중 얼마나 많은 고객이 은행을 이탈했는지 확인했습니다.

분석 결과

* 전체 고객 : **10,000명**
* 이탈 고객 : **2,037명**
* 유지 고객 : **7,963명**
* 전체 이탈률 : **20.37%**

약 5명 중 1명이 은행을 이탈한 것으로 나타났으며, 이후 분석에서는 어떤 특성을 가진 고객에게서 이러한 이탈이 많이 발생하는지를 확인했습니다.

---

## 국가별 분석

국가별 고객 수와 이탈 고객 수를 비교하여 국가에 따른 고객 이탈률을 분석했습니다.

### 고객 수

* France : 5,014명
* Germany : 2,509명
* Spain : 2,477명

국가별 이탈률을 비교한 결과

* Germany가 가장 높은 이탈률을 보였습니다.
* France와 Spain은 Germany보다 낮은 이탈률을 보였습니다.

즉, 국가에 따라 고객 이탈 패턴이 다르게 나타났으며, 특히 독일 고객을 대상으로 한 고객 유지 전략이 필요하다는 점을 확인할 수 있었습니다.

---

## 성별 분석

성별에 따른 고객 수와 이탈 고객 수를 비교했습니다.

### 여성 고객

* 전체 고객 : 4,543명
* 이탈 고객 : 1,139명
* 이탈률 : **25.07%**

### 남성 고객

* 전체 고객 : 5,457명
* 이탈 고객 : 898명
* 이탈률 : **16.46%**

분석 결과 여성 고객의 이탈률이 남성 고객보다 약 8.6%p 높게 나타났습니다.

이를 통해 성별에 따라 고객 유지율에 차이가 존재함을 확인했습니다.

---

## 연령별 분석

연령을

* 20대
* 30대
* 40대
* 50대 이상

으로 구분하여 고객 이탈률을 비교했습니다.

### 분석 결과

| 연령대    |   고객 수 | 이탈 고객 |    이탈률 |
| ------ | -----: | ----: | -----: |
| 20대    | 1,641명 |  124명 |  7.56% |
| 30대    | 4,346명 |  473명 | 10.88% |
| 40대    | 2,618명 |  806명 | 30.79% |
| 50대 이상 | 1,395명 |  634명 | 45.45% |

연령대가 높아질수록 고객 이탈률이 증가하는 경향을 확인했습니다.

특히 50대 이상 고객은 절반에 가까운 수준의 높은 이탈률을 보였으며, 20대 고객은 가장 낮은 이탈률을 기록했습니다.

이 결과를 통해 연령은 고객 이탈을 설명하는 중요한 변수 중 하나임을 확인할 수 있었습니다.

## 프로젝트 요약

이번 프로젝트에서는 SQL을 활용하여 은행 고객 데이터를 탐색하고 고객 이탈 패턴을 분석했습니다.

분석 결과,

* 전체 고객의 약 20%가 이탈했습니다.
* 독일 고객의 이탈률이 가장 높게 나타났습니다.
* 여성 고객의 이탈률이 남성보다 높았습니다.
* 연령이 높을수록 고객 이탈률이 증가하는 경향을 확인했습니다.

이번 분석을 통해 GROUP BY, CASE WHEN, COUNT, SUM, AVG, HAVING 등의 SQL 문법을 활용하여 실제 데이터를 분석하는 경험을 쌓을 수 있었으며, 데이터 탐색부터 결과 해석까지의 과정을 수행했습니다.
